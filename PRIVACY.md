# Privacy Policy

**Last updated: 2026-06-26**

## Data collection

Temporal Reasoning collects no data. All graph data is stored in a single local file (`memory.graph`) in your project directory. No data is transmitted to Minigraf or any third-party server by this plugin.

## LLM-based extraction (optional)

If you enable LLM-based extraction by setting `MINIGRAF_EXTRACTION_STRATEGY=llm`, code snippets from your project are sent to Anthropic's API using your own API key. This data is subject to [Anthropic's privacy policy](https://www.anthropic.com/privacy). This feature is disabled by default (`MINIGRAF_EXTRACTION_STRATEGY=heuristic`).

## Telemetry

There is no telemetry, analytics, or usage tracking of any kind.

## Contact

For privacy-related questions, open an issue at [github.com/project-minigraf/temporal_reasoning](https://github.com/project-minigraf/temporal_reasoning/issues).
